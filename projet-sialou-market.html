Projet : Site Vitrine Sialou Market

← Retour aux projets

Aperçu du projet

Description du projet

Ce projet est le site vitrine pour Sialou Market, une épicerie proposant une variété de produits. Le site a été conçu pour être moderne, responsive et offrir une expérience utilisateur agréable grâce à des animations fluides.

Liens

Voir le site en ligne Voir le dépot github

Fonctionnalités

  • Design Responsive : Entièrement adaptable sur mobiles, tablettes et ordinateurs de bureau.
  • Navigation Multi-pages : Structure claire avec une page d'accueil, une page produits et une page de contact.
  • Transitions de Page Fluides : Animations JavaScript pour une expérience utilisateur améliorée.
  • Catalogue de Produits : Présentation des différentes familles de produits avec images et descriptions.
  • Formulaire de Contact Fonctionnel : Double approche avec une intégration externe (FormSubmit.co) et un script backend en PHP.

Technologies utilisées

HTML5 CSS3 (Flexbox, Grid) JavaScript (ES6)
