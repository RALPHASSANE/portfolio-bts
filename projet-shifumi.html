Aperçu du projet

Description du projet

Ce projet est une implémentation web moderne du jeu classique "Pierre-Feuille-Ciseaux" (Shifumi), développée en utilisant uniquement les technologies front-end fondamentales : HTML, CSS et JavaScript. L'application propose une expérience de jeu fluide et dynamique contre une intelligence artificielle, avec un design soigné et entièrement responsive.

Fonctionnalités

  • Partie contre l'ordinateur : Affrontez une IA qui effectue un choix aléatoire à chaque manche.
  • Système de score : La partie se joue en "premier à 10 points". Le score est mis à jour en temps réel.
  • Boucle de jeu automatique : Une fois la partie terminée, le vainqueur est annoncé et une nouvelle partie démarre automatiquement après un court instant.
  • Design "Néon" Immersif : Une interface sombre avec des couleurs vives pour une ambiance "gaming" moderne.
  • Entièrement Responsive : L'expérience de jeu est optimisée pour tous les appareils, des ordinateurs de bureau aux smartphones.
  • Micro-interactions : Des animations et transitions CSS subtiles pour un retour visuel satisfaisant.

Technologies utilisées

HTML5 CSS3 (Flexbox, Animations) JavaScript (ES6+)

Structure du Code et Apprentissages

Le code JavaScript est découpé en fonctions claires et dédiées, ce qui a été un excellent exercice de structuration :

  • demarrerJeu() : Initialise ou réinitialise les scores et l'interface.
  • jouerManche() : Gère le déroulement d'un tour de jeu.
  • verifierResultat() : Compare les choix et met à jour les scores.
  • terminerPartie() : Gère la fin de partie et lance le redémarrage automatique.

Ce projet a permis de renforcer la maîtrise de la manipulation du DOM, la gestion des événements et l'implémentation d'une logique de jeu complète en JavaScript.
