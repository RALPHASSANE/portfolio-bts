Description et Fonctionnalités

Weather Time est une application web front-end qui utilise l'API de géolocalisation du navigateur pour récupérer les coordonnées de l'utilisateur. Ces coordonnées sont ensuite utilisées pour interroger l'API OpenWeatherMap et afficher en temps réel la météo locale.

Fonctionnalités clés :

  • Géolocalisation en un clic : Un simple bouton pour lancer la détection de la position.
  • Affichage des coordonnées : Affiche la latitude, la longitude et la précision.
  • Météo actuelle : Présente la ville, la température, une description et une icône.
  • Gestion des états : Affiche des messages clairs pendant le chargement ou en cas d'erreur.

Technologies utilisées

HTML5 CSS3 JavaScript (ES6+) 🌐 API Web (Geolocation, Fetch) ☁️ API OpenWeatherMap Git & GitHub

Défis et Apprentissages

Ce projet a été l'occasion de mettre en pratique plusieurs concepts clés du développement web :

  • Gestion de l'asynchronisme : Utilisation de `fetch` avec les Promises (`.then()` et `.catch()`) pour enchaîner les appels API.
  • Interaction avec les API : Lecture de documentation, construction d'URL de requête avec des paramètres dynamiques et traitement de la réponse JSON.
  • Amélioration de l'Expérience Utilisateur (UX) : Implémentation d'un conteneur de statut pour informer l'utilisateur du chargement ou des erreurs.
  • Résolution de problèmes (Versioning) : Confrontation aux défis de configuration de Git/GitHub (erreur 403) et apprentissage de l'utilisation d'un Personal Access Token (PAT).
