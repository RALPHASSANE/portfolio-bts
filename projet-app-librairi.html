Description du projet

Ce projet est une application web simple de gestion de livres, permettant aux utilisateurs de s'inscrire, de se connecter, et de gérer une liste de leurs livres favoris. L'application est développée en PHP avec une base de données MySQL et utilise PDO pour les interactions avec la base de données. L'interface utilisateur est stylisée avec CSS pour une expérience moderne et réactive.

Fonctionnalités

  • Inscription des utilisateurs : Création de comptes avec nom d'utilisateur, e-mail et mot de passe sécurisé.
  • Connexion des utilisateurs : Accès sécurisé aux comptes enregistrés.
  • Gestion des favoris : Consultation et suppression de livres de la liste personnelle.
  • Navigation : Barre de navigation pour accéder aux différentes sections de l'application.
  • Interface Utilisateur : Design simple, moderne et réactif avec CSS.

Technologies utilisées

PHP MySQL/MariaDB PDO HTML5 CSS3

Structure du Code

Le projet est organisé avec des fichiers dédiés pour la gestion de la base de données, l'authentification des utilisateurs et les différentes pages de l'application :

  • database.php : Gère la connexion à la base de données MySQL en utilisant PDO.
  • user.php : Contient la classe `User` pour la logique d'authentification.
  • inscription.php : Gère le processus d'inscription des nouveaux utilisateurs.
  • mes_favoris.php : Affiche et gère la liste des livres favoris de l'utilisateur.
  • style.css : Fichier de style CSS pour l'ensemble de l'application.
  • Autres fichiers (index.php, accueil.php, profil.php, logout.php) : Gèrent les pages principales de l'application.

Ce projet a permis de renforcer la maîtrise du développement web côté serveur avec PHP et MySQL, la gestion des sessions utilisateurs et l'interaction avec une base de données via PDO.
